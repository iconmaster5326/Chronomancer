Changelog

v1.4.0

  • Added mana cost and skill family to skill tooltips. Note that the current dataset does not yet contain cooldown or skill tag information; I hope to fix that soon.
  • Added the concept of item and character level. Note that item level currently does not correctly affect the values of base enchantments (that is: health, mana, damage).
  • Fixed issue where you could put multiple of the same enchant on an item.
  • Fixed the favicon being the default angular.js one.
  • Fixed issue with item enchant colors not rendering correctly after loading from a build.

v1.3.0

  • Added the ability to generate a link to the builds you make. They are not permalinks; they will not reflect changes you make after you get the link to the build!

v1.2.0

  • Added build importing and exporting. Right now it only imports and exports to a format local to Chronomancer; importing from Chronicon save files is a planned feature.
  • The build you're currently working on will now be automatically saved and brought back up when reloaded.

v1.1.0

  • Added this changelog.
  • Added a loading screen.
  • Item sets now show up in tooltips.
  • The item selection dialog is now more concise, and indicates when an item is part of a set.
  • The Chronicon font should now render on any browser that doesn't install TTF fonts from Internet sources. (Which should be all of the browsers.)
  • You can now ctrl-click as well as shift-click elements. Sorry, Firefox users, for making you unable to clear selected skills there.

v1.0.1

  • Fixed rendering issues on Firefox.
  • Fixed some broken skill tooltips.
  • Items that have a base quality of Enchanted may now be generated at either Enchanted or Rare quality.

v1.0.0

  • Initial release.
